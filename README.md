# Football Manager Game

Полнофункциональный футбольный менеджер с реалистичным геймплеем.

## Особенности
- Управление командой Премьер-лиги
- Реалистичные матчи с тактиками и статистикой
- Турнирная таблица
- Система бомбардиров
- Сохранение прогресса

## Установка и запуск
```bash
pip install flask
python app.py
```

Затем откройте http://localhost:5000 в браузере.

## Как играть
1. Выберите команду
2. Просматривайте статистику и следующего соперника
3. Играйте матчи в реальном времени
4. Следите за турнирной таблицей
5. Просматривайте лучших бомбардиров

## Технологии
- Flask (Python)
- HTML/CSS/JavaScript
- Bootstrap для стилизации
