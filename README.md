# Футбольный Менеджер

Простая веб-игра-симулятор управления футбольным клубом на Flask.

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Функционал

- **Главное меню**: Начать новую игру, Загрузить игру, Об игре
- **Выбор команды**: 20 команд из Premier League
- **Страница 1**: Информация о команде, следующая игра, управление
- **Страница 2**: Турнирная таблица
- **Страница 3**: Состав команды
- **Страница 4**: Финансы (доходы, расходы, баланс)
- **Страница 5**: Стадион (вместимость трибун, стоимость билетов)

## Навигация

Между страницами можно переключаться с помощью кнопок ← и → внизу каждой страницы.
